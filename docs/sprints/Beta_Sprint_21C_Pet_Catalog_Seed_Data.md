# Sprint 21C - Pet Catalog Seed Data

Precarga especies y razas/tipos para permitir pruebas reales del registro de mascotas.

## Incluye

- Seeder automático al iniciar API.
- JSON editable con especies y razas.
- Soporte para tablas `Species` y `Breeds`.
- Inserción idempotente: no duplica registros existentes.

## Especies

Perro, Gato, Ave, Pez, Conejo, Hámster, Cuy / Cobaya, Hurón, Tortuga, Reptil, Serpiente, Caballo, Mini pig, Erizo, Chinchilla.
