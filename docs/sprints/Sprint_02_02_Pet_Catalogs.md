# Sprint 02.02 - Pet Catalogs

## Definition of Done

- Build correcto.
- Swagger muestra `Catalogs`.
- `GET /api/catalogs/species` responde.
- `GET /api/catalogs/species/{speciesId}/breeds` responde.

## Uso en Mobile

La pantalla de crear mascota debe cargar primero especies y luego, al elegir una especie, cargar las razas correspondientes.
