# Sprint 01.02 - PostgreSQL + Migración inicial

## Definition of Done

- `dotnet build` correcto.
- `dotnet ef migrations add InitialCreate` correcto.
- `dotnet ef database update` correcto.
- Base `piyi_db` creada.
- API corre.
- Swagger abre.

## Notas

Este sprint no agrega endpoints nuevos. Su objetivo es persistencia real.
