# Sprint 21B - Stabilization

## Objetivo

Reparar el estado inconsistente causado por hotfixes acumulados y dejar Piyí estable para beta.

## Incluye

- Imports globales corregidos.
- Back button global para pantallas internas.
- `PiyiBackButton` centralizado.
- `PiyiPageScaffold` sin dependencia de `go_router` dentro del package `piyi_ui`.
- Registro con teléfono internacional.
- Limpieza de imports duplicados en `register_screen`.
- Logo final e íconos regenerados.
- Nombre `Piyí` corregido.
- Pubspec estabilizado.
