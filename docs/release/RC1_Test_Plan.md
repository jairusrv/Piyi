# Piyí RC1 Test Plan

## Flujo 1: Registro

1. Abrir app limpia.
2. Crear cuenta.
3. Verificar teléfono con bandera y código.
4. Confirmar que entra al Home.
5. Confirmar que Home muestra el nombre registrado.

## Flujo 2: Login

1. Cerrar sesión.
2. Entrar con email y contraseña.
3. Cerrar app.
4. Abrir app.
5. Confirmar sesión persistente.

## Flujo 3: Mascotas

1. Crear mascota.
2. Seleccionar especie.
3. Seleccionar raza.
4. Guardar.
5. Ver detalle.

## Flujo 4: Mascotas perdidas

1. Abrir Mascotas perdidas.
2. Confirmar botón volver.
3. Crear reporte.
4. Ver lista.

## Flujo 5: Negocios / Catálogo / Mapa

1. Abrir negocios.
2. Abrir detalle.
3. Abrir mapa.
4. Confirmar que no crashea.
