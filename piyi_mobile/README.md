# Piyí - BETA Sprint 15

## Flutter Catálogo Informativo Pro

Este sprint agrega la UI móvil para buscar productos/servicios ofrecidos por proveedores Pro.

## Importante

NO es marketplace transaccional.

No incluye:

- carrito
- checkout
- delivery
- órdenes
- comisiones

Sí incluye:

- búsqueda
- precio referencial
- foto
- marca
- SKU
- código de barras
- disponibilidad
- contacto directo con proveedor
- llamar
- WhatsApp
- cómo llegar
