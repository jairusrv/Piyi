# BETA Sprint 13 - Navegación profesional

## Problema

El usuario entra a pantallas como Mascotas Perdidas y no tiene forma clara de devolverse.

## Solución

- PiyiBackButton
- PiyiPageScaffold
- PiyiNavigationHelper
- Parches para pantallas principales

## Definition of Done

- Todas las pantallas internas tienen botón volver.
- Si no hay pantalla anterior, vuelve al Home.
