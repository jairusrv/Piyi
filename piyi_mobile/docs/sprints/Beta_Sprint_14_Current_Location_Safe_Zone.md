# BETA Sprint 14 - Ubicación actual

## Objetivo

Eliminar latitud/longitud manual para usuarios normales.

## Uso

- Reportar mascota perdida.
- Reportar avistamiento.
- Activar zona segura.
- Futuras alertas geolocalizadas.

## Definition of Done

- App pide permiso de ubicación.
- Usuario puede usar ubicación actual.
- Pantallas dejan de depender de coordenadas manuales.
